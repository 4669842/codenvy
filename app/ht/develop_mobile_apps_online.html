Develop Mobile Apps Online Using HTML5

Codenvy addresses multiple needs in the end-to-end cloud development workflow. And the hottest area of cloud development today is mobile apps. Developers virtually "live" in the editor/IDE, so having a complete and powerful Online HTML5 and CSS Editor (in addition to multiple other web languages) is essential to project success. Unlike device specific mobile SDKs, Codenvy can help develop mobile apps online using the new universal standard: HTML5. While some device specific hardware controls may require device SDKs, most mobile apps are actually better when developed in pure HTML5. Why? Mobile SDKs are often not perfect when it comes to variations across multiple vendors; HTML5 is far more standardized than mobile SDKs. In addition, HTML5 is as close as you can get to "write once, run anywhere" code for mobile apps.
